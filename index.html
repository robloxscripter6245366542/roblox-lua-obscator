Moonlight Deobfuscator

MoonVeil hides every string as a (data, key) byte pair rebuilt with a repeating XOR that never touches runtime state — so the whole string table lifts off statically. Paste the obfuscated Luau. Nothing executes.

● 100% client-side · no network · no eval

How it works

Every MoonVeil string sits in the source as end)(data, key). The decoder matches those sites and applies data[i] xor key[(i-1) mod #key + 1]. Because the cipher is symmetric and stateless, all strings — including branches that never run — come back in a single pass. On a 1.37 MB sample: 717 pairs, ~191 strings, ~20 ms.

Scope. An analysis lens — for understanding scripts before you run them and studying obfuscation. Not for stripping protection from code someone else obfuscated to guard their work.
